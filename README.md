# 🚗 Smart Parking Management System

A full-stack **Smart Parking Management System** developed using **Java, Spring Boot, Spring Data JPA, PostgreSQL and Bootstrap**. The application allows users to book parking slots while providing administrators with tools to manage parking operations efficiently.

## 📌 Features

### User Module

* User Registration & Login
* Secure Password Hashing using BCrypt
* View Available Parking Slots
* Book Parking Slots
* Booking History
* Logout

### Admin Module

* Admin Login
* Dashboard with Statistics
* View Registered Users
* View All Bookings
* Manage Parking Slots
* Release Vehicle
* Automatic Parking Bill Calculation
* Slot Availability Management

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/dhanasivashankar/SmartParkingSystem.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Configure PostgreSQL in `application.properties`.

4. Run the Spring Boot application.

5. Open:

```
http://localhost:8080
```
